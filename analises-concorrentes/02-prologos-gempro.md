## Segmentos Atendidos

O sistema GemPro/ERP possui versões customizadas para 6 segmentos:

### 1. Varejista
- Versão Standard estruturada para comércio varejista

### 2. Distribuidores e Atacadistas
- Versão Standard para comércio atacadista

### 3. Autopeças
- Recursos adicionais para operações típicas do comércio de autopeças

### 4. Auto e Moto Center
- GemPro/ERP customizado para Autopeças
- Agregado com Módulo de Serviços Automotivos

### 5. Assistência Técnica (Relevante para Renomaq)
- Versão para Prestadores de Serviços Técnicos
- GemPro/ERP customizado com Módulo de Serviços

### 6. Materiais de Construção
- Características típicas para lojas de materiais de construção e similares
